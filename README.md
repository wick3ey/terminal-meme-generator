# Dickhead Coin Terminal

## Project info

A terminal-style interface for Dickhead Coin, the most based token on Solana.

## Technologies used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

To run this project locally:

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```
